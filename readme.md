

BubbleMenu
==========

Objectif :
----------

Réussir à implementer un BubbleMenu dans le langage de son choix

Réussi :
--------

 - Choix du mode en maintenant le clic enfoncé ou non
 - Menu classique lorsque l'on clic normalement
 - Bulle apparaissant lorsque l'on maintient le clic enfoncé
 - Sélection d'un menu et d'un sous menu à l'aide de la bulle
 - Menu apparaissant à l'emplacement du curseur lors d'un clic droit

Spécification :
-------------

 - Langages utilisés : JavaScript - html
 - Librairie utilisée : Angular JS
 - Le fichier app.js contient l'ensemble du code nécessaire a l'utilisation du menu
